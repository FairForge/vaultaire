02How it works

You upload. The engine handles placement.

One S3 endpoint in front of hot storage and tape libraries — placement, migration, and retrieval are the engine's problem, not yours.

DAY 0–30

Hot

Performance storage on enterprise S3 infrastructure. Sub-30 ms reads, full-speed multipart uploads.

DAY 30–90

Optimized

Objects you stop touching migrate to cheaper storage with free egress. Still online, still fast.

DAY 90+

Archive

Cold data lands on tape libraries for maximum durability at archive economics.

Access any object anytime — the engine serves from the fastest available copy. You never configure lifecycle policies unless you want to.

quickstart — works in under a minute

rclone config create stored s3 \
  provider=Other \
  access_key_id=YOUR_ACCESS_KEY \
  secret_access_key=YOUR_SECRET_KEY \
  endpoint=https://stored.ge \
  region=us-east-1

rclone copy ~/backups stored:my-bucket -P

Prefer restic? restic -r s3:https://stored.ge/my-bucket init — aws-cli, s3cmd, Cyberduck, and anything else that speaks S3 works too.

04Straight answers

The parts most storage pages won't tell you.

More in the full FAQ.

Is there an SLA?

Not a contractual one yet — we won't sell one we can't back. The target is 99.5%+, the status page is real, and incident reports are written by a human. SLA credits ship with the second site.

Where does my data actually live?

On established storage providers — enterprise S3 infrastructure for hot data, professional tape libraries for archive. The control plane runs on our own hardware with tested daily off-site backups; your data doesn't depend on that one box.

How fast are cold reads?

Data that's aged to tape spools to cache on first read — seconds to a couple of minutes to first byte, longer for multi-TB restores. Hot data is <50 ms. Need everything hot forever? That's the Performance tier.

What if you disappear?

It's standard S3 — rclone sync your data out any time, free, no proprietary format. Documented wind-down commitment: minimum 60 days notice and free unlimited egress. The engine is open source, so the software outlives us too.
